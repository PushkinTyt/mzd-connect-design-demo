// component  — строка: "common", "system", или this.uiaId (имя приложения)
// component  — string: "common", "system", or this.uiaId (app name)
// eventName  — строка: идентификатор события
// eventName  — string: event identifier
// params     — объект: { payload: { ... } }  или null / не передавать
// params     — object: { payload: { ... } }  or null / omit
// vuiFlag    — boolean: true = вызвано голосом (VUI), по умолчанию false
// vuiFlag    — boolean: true = triggered by voice (VUI), omit = false

// WebSocket → ws://127.0.0.1:2700  protocol: ihu-guiifm-protocol
{
  "msgType":          "event",
  "uiaId":            component,
  "eventId":          eventName,
  "params":           params || {},
  "fromVui":          vuiFlag || false,
  "currentUiaId":     "",
  "currentContextId": ""
}

component: "common" — Global.*

Универсальные события, понимаемые любым приложением. Работают из любого контекста GUI.

Universal events understood by every app. Can be sent regardless of current context.

eventNameparamsОписание · Влияние · Где используетсяDescription · Effect · Usage
Global.GetStartupSettings— Запускает инициализацию GUI. MMUI отвечает переходом в контекст Applications / HomeScreen. Вызывается автоматически при старте GuiFramework.init() через _getStartupSettings(). Bootstraps GUI. MMUI responds with ctxtChg → Applications / HomeScreen. Called automatically on GuiFramework.init().
Global.InitGui— Альтернативный триггер инициализации. Вызывается если GetStartupSettings уже был обработан, но GUI ещё не отрисован. Alternative init trigger. Used when startup settings are already loaded but GUI is not yet rendered.
Global.IntentHome— Переход на главный экран (HomeScreen). Сбрасывает стек навигации. Используется кнопкой «Home» на рулевом колесе и в любом приложении для возврата в главное меню. Navigate to HomeScreen, clears nav stack. Triggered by the steering wheel Home button and any app's "go home" action.
Global.GoBack— Шаг назад в стеке навигации. Каждое приложение регистрирует обработчик — MMUI отправляет предыдущий контекст. Используется кнопкой «Back» и жестом назад на мультиконтроллере. One step back in the navigation stack. MMUI pops the context stack and sends the previous context. Triggered by the Back button / multicontroller.
Global.Cancel— Отмена текущего диалога или ввода. Закрывает popup без применения изменений. Аналог кнопки «Отмена». Cancel current dialog / input without applying changes. Equivalent of the Cancel button in any popup.
Global.Yes— Подтверждение в диалоге. Применяет действие (удаление, сброс, отправка). Используется во всех подтверждающих диалогах. Confirm action in dialog (delete, reset, send). Used in all confirmation dialogs.
Global.No— Отклонение в диалоге. Закрывает диалог без действия. Dismiss dialog without action.
Global.Play— Начать воспроизведение. Все медиаприложения слушают это событие через общий контроллер. Start playback. All media apps handle this via the common media controller.
Global.Pause— Приостановить воспроизведение. Активное медиаприложение реагирует на паузу. Есть задержка 250 мс в некоторых приложениях. Pause playback. Active media app responds. Some apps apply a 250 ms debounce.
Global.Resume— Возобновить воспроизведение после паузы. Аналогично Pause — 250 мс debounce в некоторых приложениях. Resume playback after pause.
Global.Next— Следующий трек / элемент. Вызывается кнопкой «⏭» на рулевом колесе или мультиконтроллере. Next track / item. Triggered by the skip-forward button on the steering wheel.
Global.Previous— Предыдущий трек / элемент. Короткое нажатие кнопки «⏮». Previous track / item. Short press of the skip-back button.
Global.NextHoldStart— Удержание кнопки «Следующий» — начало режима перемотки вперёд. Приложение переключается в FF-режим пока не придёт NextHoldStop. Long-press next → fast-forward mode start. App enters FF until NextHoldStop arrives.
Global.NextHoldStop— Отпускание после удержания «Следующий» — конец перемотки вперёд. Release after long-press next → end fast-forward.
Global.PreviousHoldStart— Удержание кнопки «Предыдущий» — начало режима перемотки назад (RW). Long-press previous → rewind mode start.
Global.PreviousHoldStop— Отпускание после удержания «Предыдущий» — конец перемотки назад. Release after long-press previous → end rewind.
Global.FastForward— Быстрая перемотка вперёд (используется голосовыми командами VUI или рулевым колесом без удержания). Fast-forward. Used by VUI voice commands or steering-wheel without hold.
Global.Rewind— Быстрая перемотка назад. Rewind.
Global.AddToFavorites— Добавить текущий элемент в «Избранное». Радиостанция, контакт или POI-точка навигации. Приложение сохраняет текущий контекст в список избранного. Bookmark current item (radio station, contact, nav POI). App saves the current context item to the Favorites list.
Global.IntentSettingsTab { payload: { settingsTab: "…" } } Открыть конкретную вкладку настроек. MMUI переключает контекст на нужный раздел без прохождения через главное меню. Используется из любого приложения для перехода в настройки. Open a specific Settings tab directly. MMUI routes to the target section without going through the main menu. Used cross-app to jump into settings.

System-level MMUI controller. Handles main menu navigation and app lifecycle.

component: "diag"

Диагностическое приложение. Работает через DBus-сервис com.jci.testdiag (svcjcitestdiag.so).

Diagnostic app. Communicates via DBus service com.jci.testdiag (svcjcitestdiag.so).

eventNameparamsОписание · ВлияниеDescription · Effect
ReadDTC { payload: { testId: <n> } } Запустить диагностический тест по ID. svcjcitestdiag.so вызывает system("/jci/scripts/startSystemScripts.sh <testId>"), что запускает jci-systemDialog — оверлейное меню с диагностическими инструментами. testId=11 — подтверждён на v74. Run diagnostic test by ID. svcjcitestdiag.so calls system("/jci/scripts/startSystemScripts.sh <testId>"), launching jci-systemDialog overlay. testId=11 confirmed on v74.
ClearDTC— Очистить коды неисправностей (DTC). Стирает диагностические ошибки из памяти ЭБУ через DBus. Clear stored Diagnostic Trouble Codes from ECU memory via DBus.
DiagLoaded— GUI уведомляет MMUI, что экран диагностики загрузился и готов принимать данные. GUI notifies MMUI that the diagnostics screen has loaded and is ready for data.
ActivateJCITest— Войти в режим JCI-тестирования. Активирует расширенные диагностические функции производителя. Enter JCI factory test mode. Activates extended manufacturer diagnostic functions.
DeactivateJCITest— Выйти из режима JCI-тестирования. Exit JCI factory test mode.
EndTest— Завершить текущий активный тест. End the current active test.
Deactivation— Завершить сеанс диагностики, освободить ресурсы. Close the diagnostics session and release resources.

component: "audiosettings"

Настройки звука — эквалайзер, громкость, система Bose, сигналы. Параметры эквалайзера передаются через динамическое поле appData: элемент списка настроек содержит поле appData, которое используется как eventName в sendEventToMmui().

Audio settings — equaliser, volume, Bose system, beeps. EQ parameters use a dynamic appData field: each settings list item has an appData field that is passed as the eventName in sendEventToMmui().
